Log Commands
1)	Commit list in details= git log
2)	Commit list in one line= git log --oneline
3)	List of all the authors within the project= git shortlog -s -n
4)	Commits list by author= git log --author <Author Name>
    OR â€“ git log --author=<2/3 starting letters>
5)	Logs of specific file= git log --author <Author Name> <filename>
6)	Logs for a specific file= git log -p <fileName>
7)	git log --all --decorate --oneline --graph
8)	git log --graph --oneline
9)	Details of a single log= git show <comitid> --stat
10)	View last 3 logs= git log -3
11)	git log --stat --oneline
12) git log --since="2013-11-12" --until="2013-11-12"

Renameing a file from git
   1) git mv <old name> <new name>

View the list of configuration
1)	git config --global --list

Alias:
1)	git config --global alias.br branch
2)	git config --global alias.ci commit
3)	git config --global alias.st status
4)	git config --global --unset "alias.name"
5)	git config --global --unset-all

Applicable only to modified file:
git commit -am "message"

Diff commands:
1) git diff <old commit> <new commit>
2) Modified VS Commited = git diff <filename>
3) Staged VS Commited = git diff --staged <Filename>
